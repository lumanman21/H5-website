IoT-Energiemanagement

Die Acrel IoT Plattform basiert auf dem Datenzentrum des IoT (Internet der Dinge), implementiert einheitliche vor- und nachgelagerte Datenstandards und bietet eine Plattform mit Datendiensten für das IoT. Die Benutzer müssen nur das Acrel IoT Power Instrument und den Sensor kaufen, ein Gateway auswählen und den QR-Code scannen, dann können sie ihre Mobiltelefone und Computer verwenden, um die erforderlichen Industriedatendienste zu erhalten. Außerdem gibt es eine Funktion zur Erstellung von Energieberichten mit einem Klick, die einen kurzen Überblick über den aktuellen Energieverbrauch bietet.

  • Struktur

    Das Acrel IoT EMS kann alle Acrel-Produkte verbinden. APM , PZ , AMC Serie für die Leistungsüberwachung, ADL Serie für das Energiequalitätsmanagement, ATC600, ARTM-PN für die drahtlose Temperaturmessung, etc. Alle Daten der Produkte werden auf zwei Arten auf die Plattform hochgeladen:

    · Der regul äre Stromzähler ist mit dem Gateway verbunden, und alle Daten werden über das Gateway auf die Plattform hochgeladen. Die Kommunikation umfasst 4G und WIF, etc.

    · Der IOT Zähler ist direkt mit der Plattform über 4G, WIFI, etc. verbunden.
